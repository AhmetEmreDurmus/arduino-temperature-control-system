# Arduino Temperature Control System

Arduino-based temperature control system using DHT11 sensor, LED and fan.

---

# Features

- Temperature monitoring with DHT11
- Open-loop LED control
- Closed-loop fan control
- Real-time temperature response

---

# Components

- Arduino Uno
- DHT11 Temperature Sensor
- LED
- DC Fan
- Transistor
- Breadboard
- Jumper Wires

---

# Control Logic

## Open Loop Control
The LED blinks independently from sensor feedback.

## Closed Loop Control
The fan automatically turns on when temperature exceeds 27°C.
